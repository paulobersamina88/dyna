Structural Dynamics Explorer - Streamlit App

Files:
1. streamlit_structural_dynamics_app.py
2. requirements.txt

How to run locally:
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run streamlit_structural_dynamics_app.py

What the app does:
- SDOF natural frequency, circular frequency, period, and static deformation
- MDOF shear-building modal analysis
- Shows mass and stiffness matrices
- Shows natural frequencies and periods
- Shows mode shapes
- Shows static total deformation under user-defined lateral load vector

Recommended future upgrades:
- damping ratio input
- time history response
- response spectrum calculator
- modal superposition of dynamic response
- animation of mode shapes
